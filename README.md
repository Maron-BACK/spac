# 스팩 청산가치 트래커

상장 스팩(SPAC)의 예치이율과 청산 예상금을 자동으로 계산해 보여주는 대시보드.

## 무엇을 하는 도구인가

- **KRX**: 현재 상장 스팩 자동 동기화 (신규 추가/폐지 마킹)
- **DART**: 증권발행실적보고서에서 납입기일, 예치/신탁계약 변경공시에서 연차별 이율 추출
- **KSFC**: 한국증권금융 12개월 거치금 금리를 보수적 fallback으로 적용
- **청산 수익률**: 납입기일 + 1050일 만기 가정, 3년차는 6개월만 반영(보수적)
- **자동 폐지**: 청산예상일 경과 종목은 자동으로 활성 목록에서 제외

## 직접 호스팅하려면

1. 이 repo를 Fork (또는 새로 만들기)
2. share.streamlit.io 로그인 → New app → 이 repo 선택
3. Settings → Secrets에 추가:
   ```toml
   IS_WEB = true
   ```
4. Deploy 클릭

자세한 단계별 가이드는 `DEPLOY.md` 참고.

## 로컬 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 사용 방법

1. 좌측 사이드바에서 **OpenDART API 키** 입력 (https://opendart.fss.or.kr 에서 무료 발급)
2. **🚀 전체 새로고침** 클릭 (1~3분 소요)
3. 종목 리스트에서 청산예상일·연환산 수익률 확인
4. 필요시 **💰 KSFC 금리** 섹션에서 12개월 거치금 금리 직접 입력
